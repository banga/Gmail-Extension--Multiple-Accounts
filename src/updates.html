Google Multi-Account Mail Checker has been updated!

  • Support for multiple Google Apps Domains has been added. Go to Options to configure.
  • Some of the issues or lack of features are due to a limitation of the Gmail RSS feeds that this extension uses. These issues occur because:
    • Gmail assigns numbers starting from 0 to your accounts and this order is fixed. If you give credentials for account 2 in the input for account 1, Gmail can show the inbox for either of the two and this extension has no way of knowing. You can find this number in your Gmail URL as https://mail.google.com/mail/u/<number>/#inbox
    • The Gmail API is limited, so features like delete, archive or mark-as-read cannot be supported as much as I want them to.
    • The API does allow viewing unread mails in only specific labels. This feature will be added in an upcoming version.
